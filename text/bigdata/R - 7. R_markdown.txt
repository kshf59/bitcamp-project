* R markdown *

1. 신뢰성있는 보고서 만들기
   - 동일한 분석과정 수행 
   - 동일한 분석결과의 재현성(reproducibility) 확보
   - 분석과정의 상세한 기술
2. 데이터 분석결과의 전달 -> 분석보고서 작성
   - 수행코드
   - 수행결과물
   - 설명글
3. 독자가 분석과정을 명확히 이해가능
4. 독자가 직접 코드를 수행하면서, 동일한 결과(재현성, reproducibility)
   의 도출여부 확인 및 코드 활용
5. 데이터 분석 전과정을 담은 보고서 작성가능
6. HTML, PDF, WORD 등의 포맷으로 export 가능

7. 수행방법
  가. File > New File > R markdown --> 문서생성창 open(New R markdown)
  나. 필요한 패키지 자동 설치 -> caTools, biTops
  다. 제목, 저자, 문서포맷(WORD,PDF,HTML) 설정 > OK --> 1번패널에 문서tab 열림
     * PDF 지원을 위해, 새로운 프로그램(Tex) 설치 필요 *
     - 윈도우용: https://miktex.org/download 접속 & 다운로드 & 설치
     - Mac용: https://tug.com/mactext 접속 & 다운로드 & 설치
  라. 열린 문서탭을 파일로 저장(디폴드: 프로젝트 폴더, 확장자: *.rmd)
  마. 각종 도구 아이콘 확인
     - 'knit' 버튼 : 3번 패널 'Viewer' 탭에 편집문서 보여주기
     - 'insert' 버튼 : R code snippet 입력

8. R markdown 편집 문법
  - # 1단계 제목
  - ## 2단계 제목
  - ### 3단계 제목
  - #### 4단계 제목
  - (http://www.naver.com) or <http://www.naver.com> : 하이퍼링크
  - *특수문자* 	: 기울임체
  - **특수문자** 	: 강조체
  - ~~특수문자~~	: 취소선
  - code chunk 입력 : <cntrl> + <alt> + i <--- ****
